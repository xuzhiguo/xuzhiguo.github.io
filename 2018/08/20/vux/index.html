vux 入门

xuzg 发表于 2018-08-20

前言

VUX 是基于WeUI和Vue(2.x)开发的移动端UI组件库，主要服务于微信页面。这是官网描述，从描述中可以知道几点：首先这是一个移动的端UI组件库，PC端的话pass掉吧。其次基于vue，那你肯定需要vue的基础了，那就开始吧。
tips：你也可以先去官网看看 –> 点这里;

安装

1、在项目根目录打开命令行，用 npm 或 cnpm 安装 vux

2、安装 vux-loader，安装这个并不是用它完全替代 vue-loader，它俩是配合使用的，后面你要使用一些插件，所以必须要安装

3、安装 less 和 less-loader，安装这俩是为了后面我们自定义一些组件的属性用的，装上装上~

4、看一下我们的 package.json

属性

这里以 XButton 为例

名字 类型 默认值 说明
type string default 按钮类型，可选值为 default,primary,warn
mini boolean false 是否为mini类型，即小尺寸的按钮
text string 按钮文字，同默认slot
action-type string button的type属性，默认为浏览器默认(submit)，可选为 submit button reset
link string vue-router 路由, 值为 BACK 等同于 go(-1)

…

属性的使用一般就直接把名字写在组件中，在指定你想要的值就可以了

样式变量

这个相比上面两个有些麻烦，不过会用以后会很方便，如果你已执行了上面按照和配置的步骤的话，那 less 和 less-loader 应该已经装好了。那只需配置一下就行了。

  1. 首先在 assets 文件下新建个 styles 文件夹，然后在这个文件夹里新建个 theme.less 文件

  2. 我们之前在 build/webpack.base.conf.js 中增加 vux 的配置，现在要在里面使用 less-theme 这个插件

  3. 这里以 XButton 为例，自定义我们的主题

名字 默认值 描述
@button-global-border-radius 5px 圆角边框
@button-global-font-color #FFFFFF 字体颜色
@button-global-margin-top 15px 与相邻按钮的 margin-top 间隙，只有一个按钮时不生效
@button-global-height 42px 按钮高度
@button-default-bg-color #F8F8F8 default类型的背景颜色
@button-default-font-color #000000 default类型的字体颜色
@button-default-active-bg-color #DEDEDE active状态下，default类型的背景颜色
@button-default-disabled-font-color rgba(0,0,0,.3) disabled状态下，default类型的字体颜色
@button-default-disabled-bg-color #F7F7F7 disabled状态下，default类型的背景颜色

太多了就不列举完了，详情请看官网…
