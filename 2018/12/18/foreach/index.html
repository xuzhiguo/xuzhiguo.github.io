es5数组的遍历方法的使用

xuzg 发表于 2018-12-18

es5 新增的遍历方法

1. forEach
  • 说明：普通的循环，无返回值，和传统的for循环类似，在里面进行逻辑处理
2. map
  • 说明：循环对每一个项处理，返回一个新的数组
3. filter
  • 说明：循环筛选出符合条件的项，返回符合条件的项组成的数组
4. some
  • 说明：循环数组，当找出符合条件，则返回true，并停止遍历，若无符合条件则返回false。（相当于或运算）
5. every
  • 说明：循环数组，当每一项都符合条件时返回true，有一项不符合条件则停止遍历，并返回false。（相当于且运算）
6. reduce
  • 说明：循环数组，累加每一项，返回和。
