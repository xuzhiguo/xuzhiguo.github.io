git 入门

xuzg 发表于 2017-06-24

前言

github是一个面向开源及私有软件项目的托管平台。在GitHub，我们既可以免费保存我们的代码，又可以十分轻易地找到海量的开源代码。

使用之前

  1. 下载

下载网址为 https://www.git-scm.com/download/

  1. 生成 SSH

打开命令行，键入 ssh-keygen。执行后会询问你文件存储位置等一系列问题，我都是直接回车，默认就行了。然后会在 c/Users/xuzg/.ssh 这个目录下生成一个 id_rsa.pub 和 id_rsa。

  1. 把你的github账户关联SSH密钥
    打开你的 github，点头像 -> setting -> SSH and GPG keys -> New SSH key，然后输入title再把 id_rsa.pub 中的内容粘贴到 key 中保存就行了。

开始使用

上面把你的git账户和你的电脑关联起来了，现在可以把你的远程仓库和本地项目用起来了。

  1. 首先你可以上github新建一个远程仓库，然后你可以进入你新建的仓库中拿到地址
  2. 在你本地的项目的根目录初始化一个git仓库，打开命令行，键入git init，填写一些信息后，最后会在那个目录生成一个 .git 文件夹

  3. 右键打开一个git命令行，设置你的用户名和邮箱

  4. 与你的远程仓库建立连接

  5. 创建一个.gitignore文件，这个文件可以让你忽略不必要提交的文件。

    tips：gitignore忽略文件的一些规则语法：

  6. 添加所有修改的文件到版本管理

  7. 提交所有添加的文件，加上提交信息。写提交信息是个好习惯，这样别人可以看提交信息知道你这次提交了什么。

  8. 上传代码及快速合并

到这一次完整的流程就走完了~
